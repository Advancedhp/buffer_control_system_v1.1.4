# Модуль оценки и непрерывного контроля буферных запасов для ROP

Локальное Streamlit-приложение для демонстрации алгоритма оценки буферных запасов в производственной системе с ресурсом, ограничивающим производство.

## Быстрый запуск Windows

```bat
python -m venv .venv
.venv\Scripts\activate
python -m pip install --upgrade pip setuptools wheel
python -m pip install -r requirements.txt
run_app.bat
```

## Быстрый запуск macOS

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip setuptools wheel
python -m pip install -r requirements.txt
chmod +x run_app.command
./run_app.command
```

Если браузер не открылся автоматически, откройте вручную адрес:

```text
http://localhost:8501
```

## Основные возможности версии fix2

- три встроенных неизменяемых демо-сценария;
- ручной ввод и редактирование рабочей копии данных;
- загрузка Excel и CSV;
- расчёт A, sigma_LT, z, Bstat, Btar, Pshort, Bopt, точки заказа и Ibuf;
- статусы: критическое отклонение, предупреждение, норма, избыточный буфер;
- настраиваемые пороги статусов;
- видимая таблица итоговых коэффициентов Kcrit, Kload, Kdev;
- dashboard с фильтрами по ROP, статусу и классу критичности;
- живой сценарный анализ без обязательного нажатия кнопки;
- вкладка тестирования с разными показателями под разные тесты;
- улучшенные графики без наложения длинных статусов на столбцы;
- схемы архитектуры и процесса с показателями A, Bopt, s, Ibuf и Pshort_current;
- структурированные управленческие рекомендации и экспорт рекомендаций;
- встроенная документация по переменным, формулам, Pshort и ограничениям модели.

## Важное замечание по Pshort

В таблице результатов используются два показателя:

- `Pshort` — вероятность дефицита при оптимальном рекомендуемом буфере `Bopt`; используется в целевой функции оптимизации.
- `Pshort_current` — текущая вероятность дефицита при фактической доступной позиции `A`; используется для управленческого вывода и визуализации текущего риска.
